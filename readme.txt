To extract the browser files place the exe and notes.txt in WTTG2_Data/Resources (the folder with the "browser_assets" file)
and execute the exe. It should extract all files into Resources/sites. To use open /sites/nexus.html in a browser.

Disclaimer: This program was made in a few hours and does not aim to satisfy any quality standards.

Known Bugs:
-Keys will not be shown on ANN Book
-On a very few sites with pictures an additional small blue square, which does not indicate a key location,
 can appear for every picture
